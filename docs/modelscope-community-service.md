# 魔塔社区服务配置

本文给维护者使用，用于把 WJR量化策略MCP 发布到魔塔社区服务或 MCP 广场。

这不是客户端用户必须阅读的内容。普通用户只需要看 README 和客户端配置示例。

## 推荐发布方式

如果魔塔页面提供“从 GitHub 仓库快速创建”，建议选择该方式。

GitHub 仓库地址：

```text
https://github.com/wangjrking/wjr-quant-strategy-mcp
```

## 基础信息

| 字段 | 建议填写 |
| --- | --- |
| 英文名称 | `wjr-quant-strategy-mcp` |
| 中文名称 | `WJR量化策略MCP` |
| 简介 | `面向 MCP 客户端的量化策略资产只读查询服务，支持查看策略介绍、最新信号、历史信号和历史效果材料。` |
| 所有者 | 使用你的魔塔账号 |
| 是否公开 | 公开 |

## 服务配置

如果平台要求填写外部 MCP 服务地址：

```text
http://mcp.wjr.world/mcp-servers/wjr-quant-sse/sse
```

Transport 选择：

```text
SSE
```

认证方式：

```text
Authorization: Bearer <你的AI网关消费者Token>
```

不要在魔塔公开页面、README、截图或 Issue 中展示真实 Token。

## 推荐公开描述

```text
WJR量化策略MCP提供已发布量化策略资产的只读查询能力，包括策略介绍、最新信号、历史信号和策略历史效果材料。服务不生成新预测，不自动荐股，不构成买入、卖出或持仓建议。历史表现不代表未来收益。
```

## 工具描述

| 工具名 | 中文描述 |
| --- | --- |
| `get_strategy_overview` | 查看可用策略清单，或查看指定策略的介绍、规则说明和适用边界 |
| `get_latest_signal` | 查看某个策略已经发布的最新信号 |
| `get_strategy_history_and_performance` | 查看某个策略的历史信号和历史效果材料 |

## 发布后检查

发布后至少检查：

- 页面能看到中文简介。
- 页面能看到三个工具。
- 工具说明不是技术内部口径，而是普通用户能理解的描述。
- 外部调用示例不包含真实 Token。
- 风险提示明确说明“不构成投资建议”。

## 不建议公开的内容

不要公开：

- 真实 Token
- 数据库连接串
- 服务器内网 IP
- 内部资产目录
- 内部模型文件路径
- 未发布策略参数
- 未审计的生产信号
